什么是XML

XML是一种用来传输和存储数据的可扩展标记语言。

作用：XML用于传输和数据存储。HTML用于显示数据。

Dtd文档类型定义是一套为了进行程序见的数据交换而建立的关于标记符的语法规则。

DTD是不是xml文档的一部分，它总是根元素的定义之上

结构

所有的 XML 文档（以及 HTML 文档）均由以下简单的构建模块构成：

  • 元素

    • 元素是 XML 以及 HTML 文档的主要构建模块，元素可包含文本、其他元素或者是空的

  • 属性

    • 属性可提供有关元素的额外信息

    • 如：

  • 实体

    • 实体是用来定义普通文本的变量

  • PCDATA

    • PCDATA 的意思是被解析的字符数据（parsed character data）
    • PCDATA 是会被解析器解析的文本，这些文本将被解析器检查实体以及标记

  • CDATA

    • CDATA 的意思是字符数据（character data）
    • CDATA 是不会被解析器解析的文本

注意：

标签区分大小写

版本是标志，xml会识别这个标志，然后将后面的内容解析成xml文本

DTD是什么？Document Type Definition——文档类型定义。

实体： 用于定义引用普通文本或特殊字符的快捷方式的变量
